# Linux_System# Linux Security Audit

## Project Overview
The Linux Security Audit project was developed as part of my Cybersecurity Internship. This project focuses on analyzing the security configuration of a Linux system, identifying common security risks, and applying basic hardening techniques to improve system protection.

## Objectives
- Perform a basic Linux security audit
- Review user accounts and permissions
- Check firewall configuration
- Improve system security awareness
- Learn cybersecurity best practices

## Tools Used
- Linux Terminal
- chmod
- UFW Firewall
- systemctl

## Features
- File Permission Analysis
- User Account Security Review
- Firewall Status Verification
- Basic System Hardening
- Security Documentation

## Learning Outcomes
- Linux System Administration
- Access Control Management
- Firewall Configuration
- Security Auditing
- Cybersecurity Fundamentals

## Conclusion
This project enhanced my practical knowledge of Linux security and cybersecurity fundamentals. It provided hands-on experience in identifying security issues and implementing basic security controls using Linux tools and commands.
